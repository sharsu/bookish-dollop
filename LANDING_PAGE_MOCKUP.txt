VALIDATION & ERROR HANDLING
═══════════════════════════════════════════════════════════════════════════════

If "Start Exam" clicked without name:
  - Red error message appears below button
  - "Please enter your name"

If question count invalid:
  - Error: "Questions must be between 5 and 100"

If topic selection has insufficient questions:
  - Error: "Not enough questions available. Found: X"

Errors display in red (#dc2626) above the Start button


USER FLOW
═══════════════════════════════════════════════════════════════════════════════

1. App loads → Landing page shown
2. Child sees:
   - Welcoming title "Maths Exam Prep"
   - Subtitle "Practice makes perfect!"
   - Form asking for their name
   - Option to customize quiz
   - Big green button "🚀 Start Exam"

3. Child fills form:
   - Enters name (e.g., "Alex")
   - Can adjust question count if desired
   - Can adjust time if desired
   - Can select specific topics (optional)

4. Clicks "🚀 Start Exam":
   - App validates input
   - Loads quiz with random questions
   - Starts 60-minute timer
   - Shows first question

5. If error:
   - Red message appears
   - Child can correct and retry
