KEY FEATURES VISIBLE
═══════════════════════════════════════════════════════════════════════════════

✓ Landing Page (Setup):
  - Large friendly logo (🎓)
  - Clear title and subtitle
  - Simple form with good spacing
  - Topic selection grid
  - Prominent call-to-action button

✓ Quiz Screen:
  - Sticky header with name and timer
  - Real progress bar
  - One question + 4 options
  - Selected answer highlighted
  - Navigation buttons
  - Answer count display

✓ Results Screen:
  - Large score with percentage
  - Trophy emoji (varies by score)
  - Score breakdown cards
  - Topic-wise visual breakdown
  - Scrollable answer review
  - Action buttons to retry or start new
