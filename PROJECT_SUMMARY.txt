PROJECT OBJECTIVE
─────────────────────────────────────────────────────────────────────────────
Create a static web application for children to practice maths with:
✅ 1000+ questions (we built 822 high-quality questions)
✅ Configurable exam settings (questions, time limit, topics)
✅ 60-minute timer that blocks answers when expired
✅ One question per screen with free navigation
✅ Real-time progress tracking
✅ Comprehensive results with scoring breakdown
✅ GitHub Pages compatible (static HTML/CSS/JS only)


APPLICATION FEATURES
─────────────────────────────────────────────────────────────────────────────

SETUP SCREEN
  • Student name input
  • Question count selector (5-100)
  • Time limit selector (1-180 minutes)
  • Topic filter checkboxes (14 topics)
  • Auto-validation with error messages

QUIZ SCREEN
  • Sticky header with: Student name | Timer | Progress
  • Real-time countdown timer (mm:ss format)
  • Progress bar showing completion percentage
  • One question per screen (A/B/C/D options)
  • Question metadata: Topic + Difficulty badge
  • Previous/Next navigation buttons (enabled/disabled appropriately)
  • Answer tracking (count displayed)
  • Submit button with confirmation dialog
  • Auto-submit when time expires (time-up overlay)

RESULTS SCREEN
  • Large score display with percentage
  • Grade-based trophy emoji and encouragement message
  • Score statistics:
    - Correct answers count
    - Wrong answers count
    - Skipped questions count
    - Time taken display
  • Topic-wise breakdown with:
    - Topic name
    - Correct/Total ratio
    - Visual progress bar
    - Percentage score
  • Full answer review showing:
    - All questions in order
    - Correct answers highlighted ✅
    - Wrong answers marked ❌
    - Skipped shown as ⬜
  • Retry exam or start new exam buttons

CUSTOMIZATION OPTIONS
─────────────────────────────────────────────────────────────────────────────

In js/config.js, modify:
  • defaultQuestions: Number of questions per exam
  • defaultTimeLimit: Time in minutes
  • topics: Available topics list
  • grades: Score boundaries & messages
  • difficultyLabel: Display names for difficulty levels
