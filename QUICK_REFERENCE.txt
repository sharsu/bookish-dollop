🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: App shows blank page
→ Refresh (Ctrl+R), clear cache (Ctrl+Shift+Delete)

Problem: Start button doesn't work
→ Check browser console (F12), look for red errors

Problem: Timer wrong
→ Refresh page

Problem: Questions not loading
→ Check: js/questions.js exists and is 125+ KB

Problem: Results not showing
→ Press F12, check for JavaScript errors
